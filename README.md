# Simulating-Chaos
Python simulation of the Logistic Map to derive the Feigenbaum constant ($\delta$). It uses a "burn-in" phase to remove noise and bisection search to find exact bifurcation points ($r$). By calculating the interval ratios, it demonstrates universal mathematical patterns in chaotic systems.
